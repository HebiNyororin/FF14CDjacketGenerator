# FF14 CD Jacket Generator

FF14のキャラクターデータ（DC、サーバー名、メインジョブ、プレイスタイルなど）を選択して、CDジャケット風のオシャレなキャラクターカード画像を生成できるWebアプリケーションです。

## 特徴
- **リアルなCDケースの質感**: CSSグラデーションを駆使し、透明なアクリルケース（ジュエルケース）のハイライトやヒンジ部分を再現。
- **フルカスタマイズ可能なカラー**: ジャケット背景、ディスク盤面、テキスト色、全体の背景色をカラーピッカーで自由に調整可能。
- **多彩なフォント**: Noto Sans, Noto Serif, M PLUS, Shippori, Cinzel, Playfair, Great Vibes, Cormorant, Josefin Sans など、用途に合わせて9種類のフォントから選択。
- **ワンクリック画像保存**: html2canvasを使用し、作成したCDジャケット画像をローカルにPNG形式で保存。

## 使い方
1. リポジトリをクローンまたはダウンロードします。
2. フォルダ内の `index.html` をブラウザで開きます。（ローカルファイルとしてそのまま動作します）
3. 左側のコントロールパネルから各種項目（画像、色、フォント、ジョブなど）を選択・入力します。
4. 画像位置をドラッグして微調整します。
5. 「Save CD Jacket」ボタンをクリックして画像をダウンロードします。

## 使用技術
- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)
- [html2canvas](https://html2canvas.hertzen.com/) (画像生成用)
- [Google Fonts](https://fonts.google.com/)
